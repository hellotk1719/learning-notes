## Installation

### Server Requirements

* PHP >= 7.0.0
* OpenSSL PHP Extension
* PDO PHP Extension
* Mbstring PHP Extension
* Tokenizer PHP Extension
* XML PHP Extension

### Installing Laravel

```
composer create-project --prefer-dist laravel/laravel projectName "5.5.*"
```

* [Installation - Laravel](https://laravel.com/docs/5.5/installation)
